# Mina Todos

En enkel todo-applikation byggd med React, TypeScript och Vite.  
Syftet med projektet är att öva på komponenter, props, state-hantering och enklare testning.

## Funktioner

- Lägg till nya todos
- Markera en todo som klar
- Ta bort en todo
- Todos sparas i `localStorage` så de finns kvar även efter sidladdning

## Tekniker som används

- React (funktionella komponenter och hooks)
- TypeScript
- CSS för styling
- Vite som bundler
- Vitest och React Testing Library för tester
- GitHub Actions för automatisk testkörning (CI)
- Vercel för deployment

## Kör projektet lokalt

1. Installera paket:
   npm install

2. Starta utvecklingsläget:
   npm run dev

3. Bygg projektet:
   npm run build

## Tester

Projektet använder Vitest.

Kör tester med:
npm test

## Deploya projektet

Projektet är deployat via Vercel.
